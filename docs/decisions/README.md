# Decision records

Use this directory for short records of durable decisions whose reasoning will matter
after the implementation details have changed: data ownership, security boundaries,
deployment invariants, protocol direction, and other expensive-to-reverse choices.

One decision per file, named `NNNN-short-title.md`. Include the date, status, context,
decision, and consequences. Do not use decision records as a backlog or release log.

The original mutual-aid redesign remains available as frozen historical context in
[`../archive/mutual-aid-rework-2026-07.md`](../archive/mutual-aid-rework-2026-07.md).
